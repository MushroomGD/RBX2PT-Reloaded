
# RBX2PT-Reloaded

Originally made by NonsensicalOne (https://github.com/NonsensicalOne/RBX2PT-web), RBX2PT-Reloaded is a converter that allows you to use ROBLOX clothes on Polytoria by converting. The thing with the original RBX2PT-Web, is that Polytoria now uses a new template and RBX2PT-web dosen't seem to be fixing this soon. So I made RBX2PT-Reloaded, a newer version of RBX2PT-Web that fixes the new template issue.
